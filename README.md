# contextself.com

Personal website for Xianglong Tao - systems thinking, equilibrium analysis, and software engineering.

## Tech Stack

- **Framework**: [Astro](https://astro.build)
- **Deployment**: Vercel
- **Domain**: contextself.com

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Structure

```
src/
├── layouts/
│   └── Layout.astro       # Base layout with nav/footer
├── pages/
│   ├── index.astro        # Homepage
│   ├── about.astro        # About page
│   └── projects.astro     # Projects page
```

## Design Principles

- Clean, minimal design
- System fonts only
- No JavaScript (static site)
- Fast, accessible, boring
- Engineer-focused, not marketing
